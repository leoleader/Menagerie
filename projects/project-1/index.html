Super Smash Bros. Ultimate Tournament Simulator

February 1, 2023

My latest project that I started developing over last Winter Break is an app that uses a database of tournamnet results for Smash Ultimate pro players to simulate games between them. Using a combination, of player head-to-head history, player winrate, how highly ranked the characte they play is in the meta, character matchup charts, the program tries to predict who would win between two players. So using this algorithm, users of the app will be able to make tournament brackets as big as top 16s with professional players and the program will simulate the bracket. The design will be similar to this Hunger Games Simulator https://brantsteele.net/hungergames/disclaimer.php in the way that it will provide humorous randomly generated messages on how the platers beat each other and what happened in the matches based on the characters played in the match. Back end is currently built in python but I want more familliarity with node.js so I plan on transfering it into node.js. I am using a SQLite database from smashdata.gg and am building the frontend using react.js.
