Super Smash Bros. Ultimate Tournament Simulator

My latest project that I started developing over last Winter Break is an app that uses a database of tournamnet results for Smash Ultimate pro players to simulate games between them. Using a combination, of player head-to-head history, player winrate, how highly ranked the characte they play is in the meta, character matchup charts, the program tries to predict who would win between two players. So using this algorithm, users of the app will be able to make tournament brackets as big as top 16s with professional players and the program will simulate the bracket.
